Gastro-Tag

Introdução:

Repositório do projeto Integrador Gastro-Tag para criar um sistema de software de auxilio de identificação de alimentos dentro da cozinha e deposito, o sistema funcionará através de um aparelho eletrônico que será utilizado por funcionários permitidos onde preencherão com informações necessárias sobre as condições do alimento e imprimirá a etiqueta de identificação.

Principais Funcionalidades:

•	Cadastro de Produtos:  Inserção de dados de produtos, incluindo o nome, prazo de validade original, e regras específicas de validade/armazenamento pós-abertura.
•	Geração de Etiquetas: geração de etiquetas contendo informações obrigatórias para o produto pós-aberto como data de abertura, data de descarte, nome, condição de armazenamento, lote.
•	Impressão de etiquetas: integração com um dispositivo de impressão de etiquetas geradas


Requisitos:

1.	Dispositivo móvel (tablet)
2.	Impressora térmica ou mini impressora de etiquetas configurada
3.	Banco de dados para armazenar os produtos cadastrados
